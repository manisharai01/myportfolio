# Manisha Rai — Portfolio

A polished, Apple-inspired personal portfolio built with **React + Vite + Framer Motion + Lenis**.

## Features
- Buttery-smooth scrolling (Lenis)
- Cinematic hero with parallax + text reveal
- Scroll-triggered fade/slide animations on every section
- Animated experience timeline & project cards
- Glassmorphism navbar with scroll state
- Cursor-following gradient blob
- Fully responsive (mobile → 4K)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

> Note: This is a **web app**. React Native is for native mobile apps; for a website you want React (this project). The same components can later be ported to React Native Web if needed.
